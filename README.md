# Cantilever-Beam-Structural-Fatigue-and-Modal-Analysis
This project investigates the structural behavior of a cantilever beam using analytical calculations and finite element analysis (FEA). The study was conducted as part of the MECA 332 – Mechatronics Components Design course at Istanbul Bilgi University.
The beam was modelled in SolidWorks and analyzed in ANSYS to evaluate its static response, fatigue performance, stress concentration effects, and vibration characteristics.

Objectives
Create a 3D model of a cantilever beam assembly.
Perform static structural analysis under external loading.
Compare analytical calculations with finite element results.
Investigate the effect of fillet radius on stress concentration.
Evaluate fatigue life under fully reversed loading conditions.
Determine natural frequencies and mode shapes through modal analysis.
Tools and Software
SolidWorks
ANSYS Mechanical
Mechanical Design Theory
Finite Element Analysis (FEA)
Project Scope
Static Structural Analysis

A vertical load of 4500 N was applied at the free end of the beam. Stress distribution, deformation, factor of safety, and allowable load capacity were evaluated.

Mesh Convergence Study

Different mesh sizes were tested to verify solution accuracy and ensure convergence of stress results.

Stress Concentration Investigation

The influence of fillet radius on maximum stress and deformation was analyzed for multiple geometric configurations.

Fatigue Analysis

Fatigue performance was assessed using analytical calculations based on Shigley’s fatigue design methodology and validated using ANSYS Fatigue Tool.

Modal Analysis

Natural frequencies and mode shapes were obtained to identify potential resonance risks during operation.

Key Findings
Increasing the fillet radius reduced peak stress by approximately 32%.
Analytical and ANSYS results showed strong agreement.
The fatigue safety factor remained above unity for the specified loading conditions.
The first natural frequency was identified at approximately 449 Hz.
